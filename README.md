## 🧑‍💻 Getting Started (for developers)
## Clone the repo
git clone https://github.com/shihabuddin-dev/shihabuddin-repo.git

## Navigate to the project
cd shihabuddin-repo

## Install dependencies
npm install

## Start the dev server
npm run dev

## 📦 Deployment
## To build for production
npm run build
